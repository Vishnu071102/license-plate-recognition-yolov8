# 🚗 License Plate Recognition using YOLOv8

## 📌 Overview

This project implements an automated license plate detection and recognition system using deep learning.

## 💡 Approach

* YOLOv8 for license plate detection
* OCR for character recognition
* Image preprocessing for better accuracy

## 🧠 Tech Stack

* Python
* OpenCV
* YOLOv8
* Tesseract OCR

## ⚙️ Workflow

1. Input image/video
2. Detect license plate using YOLO
3. Crop plate region
4. Apply OCR to extract text

## 📊 Results

* High detection accuracy
* Robust in varying lighting conditions


## 🚀 Future Improvements

* Real-time video processing
* Multi-camera system
* Improved OCR accuracy
